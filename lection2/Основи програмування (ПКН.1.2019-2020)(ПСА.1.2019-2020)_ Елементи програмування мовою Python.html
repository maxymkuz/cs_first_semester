    Елементи програмування мовою Python.

    Елементи програмування мовою Python.


    Розглянемо більш детально рядки модуля для обчислення значень логістичного відображення (модуль chaos.py).

    # File: chaos.py

    # A simple program illustrating chaotic behavior.

     

    def main():

        print("This program illustrates a chaotic function")

        x = float(input("Enter a number between 0 and 1: "))

        for i in range(10):

            x = 3.9 * x * (1 - x)

            print(x)

     

    main()

    Перші два рядки в цьому модулі починаються з символу #:

    # File: chaos.py

    # A simple program illustrating chaotic behavior.

    Такі рядки називають коментарями й вони призначені для тих хто буде читати програму. Інтерпретатор Python проігнорує ці рядки та відповідно не буде їх виконувати. Так само інтерпретатор буде поводитись і з частиною будь-якого рядка програми від символу # і до кінця рядка.

    Наступний рядок програми містить заголовок оголошення функції, яка має назву main :

    def main():

    Поняття функції це одне з ключових понять у програмуванні. Більш детально функції будуть вивчатися пізніше, але на одній з переваг, яку надає використання функції в цій програмі потрібно наголосити. Якщо програма розробляється в інтерактивному режимі й виконується рядок за рядком то для її повторного виконання потрібно знов повторити виконання всіх рядків (можливо, запустити новий сеанс розроблення (restart shell в IDLE) і знов повторити всі рядки. Використання функції дозволяє здійснити виконання розрахунків шляхом безпосереднього виклику цієї функції main() чи chaos.main().

    Перший рядок функції

        print("This program illustrates a chaotic function")

    вказує інтерпретатору Python вивести (надрукувати) на екран повідомлення про початок роботи програми. Всю цю роботу виконує функція print(), яку називають вбудованою функцією бо вона розроблена творцями мови Python, надається програмісту та готова до використання.

    Наступний рядок доволі складний:

        x = float(input("Enter a number between 0 and 1: "))

    x називають змінною. Змінні використовуються для того щоб можна було за їх іменем звертатися до значень у будь-якому місці програми. Цей рядок використовується для того щоб отримати дані від користувача програми. Коли Python виконує цей рядок то на екран буде виведено повідомлення з текстом, який вказано у лапках Enter a number between 0 and 1:. Після цього виконання програми буде зупинено до тих пір поки користувач щось не введе з клавіатури та не натисне <Enter>. Значення (число), яке введе користувач буде збережено у змінній x ("значення буде збережено у змінній" не зовсім точно описує те, що насправді відбувається, але поки що тут й далі в тексті залишимо це без додаткових пояснень).

    Наступний рядок програми дозволяє організувати повторне виконання певної частини програми. Таке повторне виконання частини програми називають циклом.

        for i in range(10):

    Цей цикл вказує повторно (10 разів) щось виконати. Десять разів буде виконано все те, що записано після першого рядка циклу із відступом у 4 пробіли. Ці два рядки називають тілом циклу.

            x = 3.9 * x * (1 - x)

            print(x)

    Використання циклу дозволяє уникнути повторного написання одних і тих же рядків програми. Якщо не використовувати цикл то ці два рядки потрібно буде написати десять разів.

    Розглянемо перший рядок тіла циклу. За допомогою цього рядка виконуються обчислення.

            x = 3.9 * x * (1 - x)

    Права частина цього рядка, яка знаходиться після оператора присвоювання = це математичний вираз. Символи * та - в цьому виразі це оператори множення та віднімання, відповідно. Якщо значення x дорівнює 0.25 (введено користувачем) то результат обчислення це 3.9*0.25*(1 − 0.25) або 0.73125.

    Після того як значення в правій частині рядка обчислено, воно зберігається, або часто говорять присвоюється змінній, яка знаходиться в лівій частині перед =. Потрібно звернути увагу що спочатку значенням x було 0.25, а після першого виконання цього рядка воно буде змінено на 0.73125.

    Другий рядок в тілі циклу, здається, цілком зрозумілий і за його допомогою на екран виводиться значення змінної x.

            print(x)

     

    Після того як на екран буде виведено 0.73125 ці два рядки тіла циклу будуть знову виконуватися. Тіло циклу буде виконуватися 10 разів.

    Потрібно звернути увагу що при другому виконання (другій ітерації циклу) нове значення для x буде обчислено вже на основі виразу 3.9*0.73125*(1−0.73125).

    Останній рядок модуля це виклик функції.

    main()

    Виконання цього рядка це виконання всіх рядків цієї функції. Без цього рядка модуль буде містити тільки оголошення (визначення) функції, і щоб вона виконувалася її буде потрібно додатково викликати.

     

    Дослідити роботу цієї програми зручно скориставшись можливостями, які надає сервіс http://www.pythontutor.com/. За допомогою цього сервісу можна покроково виконувати програму (ліва панель) та відстежувати значення всіх змінних (права панель).

     

     

    Попри такий детальний розгляд цієї короткої програми залишаються без відповіді багато запитань. Що означають в програмі range, float, input? Як інтерпретатор розуміє що було введено число? Як виконуються математичні операції? Спробуємо знайти відповіді на ці запитання та продовжити знайомитися з мовою програмування Python.

    Кожна мова програмування підтримує (містить) набір вбудованих типів, які дозволяють організувати роботу з даними різного виду. Також більшість мов програмування розповсюджуються разом зі стандартною бібліотекою. Стандартна бібліотека це готовий набір інструментів для розроблення програм.

    Стандартна бібліотека Python містить вбудовані типи, вбудовані функції, вбудовані константи та велику кількість модулів, які дозволяють спростити розроблення програм.

    В Python вбудовані такі прості типи: булеві значення, цілі числа, числа з плаваючою крапкою, рядки.

    Наступний приклад дозволяє пересвідчитися в цьому за допомогою використання вбудованої функції type(), яка визначає і виводить на екран інформацію про тип.

     

     

    В Python вбудовані також й інші типи: список, кортеж, множина, словник. Програмісти також можуть створювати і свої власні типи.

     

     

    В загальному поняття типу властиве й іншим елементам мови програмування. Строго кажучи ці елементи мови програмування називають об'єктами і саме об'єкти створються в програмах і з об'єктами виконуються всі дії в програмах. Змінна це об'єкт, функція також об'єкт...

     

    import math

    print(type(print))       # builtin_function_or_method

    print(type(input))       # builtin_function_or_method

    print(type(type))        # builtin_function_or_method

    print(type(range))       # type

    print(type(math))        # module

    print(type(math.sin))    # builtin_function_or_method

    print(type(main))        # function (user-defined function)

    print(type(type(65)))    # type

    print(type(Exception())) # Exception

    print(type(range(5)))    # range

    print(type(2+3j))        # complex  (complex number) (we may not see this type)

     

    В Python вбудовані наступні константи:

     

    Серед вбудованих функцій потрібно виділити ті, за допомогою яких можна створювати змінні різних типів, або здійснювати перетворення типів.

     

    Інша група вбудованих функцій дозволяє спростити виконання математичних операцій.

     

    Отже, range, float, input це вбудовані функції стандартної бібліотеки Pyhon. Рядок

    x = float(input("Enter a number between 0 and 1: ")) складається з двох частин спочатку виконується функція input() а потім результат її виклику (рядок) перетворюється за допомогою функції float() до типу float.

    Операції в Python здійснюються за допомогою операторів і наступна таблиця містить їх перелік. Детальніше вони будуть вивчатися пізніше.

     

    Призначення оператора

    Оператори

    Арифметичні

    +, -, *, /, //, **, %, - (unary), + (unary)

    Порівняння

    <, <=, >=, >, ==, !=

    Присвоювання

    +=, -=, *=, /=, //=, **=, %=, <<=, >>=

    Логічні

    and, or, not

     

    Оператори дозволяють реалізувати взаємодію між різними типами, але результат не завжди очікуваний. Не всі операції можна здійснювати при взаємодії типів, а також і з різними типами.

     

    Серед арифметичних операторів потрібно звернути увагу на оператори ділення бо їх є декілька і результати операцій за їх участю потребують додаткових пояснень.

    Оператор ділення / дозволяє виконувати звичайне ділення й у випадку ділення цілих чисел (int) буде отримано число з плаваючою крапкою (float).

     

    Оператор ділення // дозволяє виконувати цілочисельне ділення. Цілочисельне ділення в Python заокруглює результат до меншого. У випадку ділення додатних чисел результат буде відрізнятися від результату коли одне з чисел від’ємне.

     

    Оператор ділення по модулю % дозволяє отримати залишок від ділення.

     

    Обчислення здійснюють таким чином, що якщо a/b = q з остачею r тоді b*q + r = a (3*-2+2=-4).

     

    При виконанні обчислень потрібно зважати на пріоритети операторів та їх асоціативність.

    Асоціативність всіх операторів зліва-направо крім оператора піднесення до степеня, який має асоціативність справа-наліво.

    Дійсні числа представляються за допомогою типу float. Обчислення з використанням чисел з плаваючою крапкою характеризуються скінченною точністю.

     

    При порівнянні результатів таких обчислень доцільно використовувати спеціальну функцію, яка є однією з функцій модуля math стандартної бібліотеки Python.

    Модуль math містить також й інші функції та константи, які дозволяють виконувати різноманітні обчислення.
